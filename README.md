# OPC Client

## Overview

OPC Client is a C# .NET application designed to establish secure communication with OPC UA servers used in industrial automation systems.

The application will provide functionality for:

- OPC UA server discovery
- Secure server connections
- Node browsing
- Reading tag values
- Writing tag values
- Connection monitoring
- Error handling and logging

## Technologies

- C#
- .NET
- OPC UA
- Visual Studio Code
- GitHub

## Project Goals

This project was created to gain hands-on experience with industrial communication protocols commonly used in:

- PLC systems
- SCADA systems
- Historians
- Power system automation
- Industrial control networks

## Future Enhancements

- GUI dashboard

- Real-time monitoring
- Multiple OPC server support
